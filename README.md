# preyecto1ampresariales

## Backend (FastAPI)

1. Entra a la carpeta `version en python`.
2. Instala dependencias:
	- `pip install -r requirements.txt`
3. Ejecuta el servidor:
	- `uvicorn main:app --reload`
4. La API queda en:
	- `http://127.0.0.1:8001`

Credenciales de prueba creadas automaticamente:

- Usuario: `admin`
- Contrasena: `admin123`

## Frontend

Abre `index.html` en un servidor estatico (por ejemplo Live Server en VS Code) y luego inicia sesion con las credenciales de prueba.
